/fleet Build an application to create and deploy auto insurance. 

Features include: 
- Landing page that lists policies
- Policy create page that allows selection of year, make and model of vehicle, coverage and deductible selection.
- Vehicles are exotic sports cars.
- Pricing calculator feature to allow user decision before creation of the policy
- Pricing calculator uses random numbers between 5 and 500 per coverage
- Creating a policy generates a unique identifier and a six month policy term that starts tomorrow.
- Clicking a policy from the landing page shows a policy page with a read-only description of vehicles and coverages.
- An edit button on the policy page allows changing the vehicle and coverages, with a recalculation on save. Expiry dates stay the same.

Requirements Include:
- Responsive, accessible design
- Authentication with Entra
- Storage Access Keys are disabled, and managed identities are used
- Secure communication over the latest TLS is allowed but no other
- all infrastructure is managed with Terraform IaC

- A database is used for persistent storage
- The database is encrypted at rest using cloud-provided keys
- The database has backup disabled for cost reasons
- Storage quotas for database and disks are 2GB for data only
- Free SKUs are preferred; lowest cost possible
- Lives in a spoke of our hub-and-spoke network. Consult with and updated cidr.txt to get the best fit, available address space.
